# NSD Plugin

Плагин для IntelliJ IDEA, который связывает IDE с внешними инструментами через локальный HTTP-сервис и упрощает отправку Groovy-файлов во внешний сервис.

## Цель

Сделать плагин, который:
- умеет принимать запросы от браузерных расширений через локальный веб-сервис;
- умеет выполнять действия в IDEA в зависимости от текущего состояния открытых проектов;
- добавляет удобную кнопку для отправки Groovy-файла на внешний URL.

## Основные задачи

### 1. Локальный веб-сервис

Плагин должен поднимать веб-сервис на порте, заданном в конфигурации.

Поведение при получении запроса:
- если в IntelliJ IDEA открыто несколько проектов, сервис возвращает имена открытых проектов;
- если открыт только один проект, в этом проекте создается новый файл.

Сервис предназначен для вызова из браузерных расширений.

### 2. Кнопка в редакторе Groovy

В окне редактора Groovy-файлов должна появиться новая кнопка.

По нажатию на кнопку:
- текущий Groovy-файл отправляется на внешний URL;
- URL задается в конфигурации плагина.

## Конфигурация

На первом этапе в плагине должны быть предусмотрены минимум две настройки:
- порт локального веб-сервиса;
- URL для отправки Groovy-файлов.

## Результат

Плагин должен стать мостом между IntelliJ IDEA, браузерными расширениями и внешним сервисом, сохраняя простой сценарий настройки и использования.
